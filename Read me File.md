 k4k-maritime-safety-model

This repository contains the code and synthetic dataset used in the study on knock-for-knock clauses and maritime safety incentives.

Repository structure:

k4k-maritime-safety-model
│
├── generate_dataset.ipynb
├── analysis_model.ipynb
├── synthetic_maritime_dataset.csv
└── README.md


Files

generate_dataset.ipynb  
Generates the synthetic maritime accident dataset used in the analysis.

analysis_model.ipynb  
Performs the statistical analysis using the dataset and produces the results reported in the study.

synthetic_maritime_dataset.csv  
Example dataset generated using the simulation model.


Usage

1. Run `generate_dataset.ipynb` to generate the synthetic dataset.
2. Run `analysis_model.ipynb` to perform the statistical analysis.


Note

The dataset used in this study is synthetically generated for research purposes and does not represent real accident records.
